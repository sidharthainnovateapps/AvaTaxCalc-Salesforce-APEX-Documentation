AddressSvcSoap Class

The AddressSvcSoap Object is the proxy interface for the Avalara Address Service. The constructor takes one string argument, which is the service URL your method calls will reference. All Address Service APIs are methods of the AddressSvcSoap object.

Note that the embedded AddressSvc.Profile and AddressSvc.Security objects are instantiated automatically when the AddressSvcSoap Object is instantiated (no need to "new" Profile or Security).

global class AddressSvcSoap
{
// data members
     global AddressSvc.Profile Profile;
     global AddressSvc.Security Security;
// methods
     global AddressSvcSoap(String SvcURL);
     global AddressSvc.IsAuthorizedResult IsAuthorized(String Operations;
     global AddressSvc.ValidateResult Validate(AddressSvc.ValidateRequest ValidateRequest);
     global AddressSvc.PingResult Ping(String Message);
}

Example:

AVASFSDK.AddressSvc.AddressSvcSoap addressSvc =
     new AVASFSDK.AddressSvc.AddressSvcSoap('https://development.avalara.net');
addressSvc.Security.UsernameToken.Username = '[UserName/AcountNum Here]';
addressSvc.Security.UsernameToken.Password = '[Password/LicenseKey Here]';
addressSvc.Profile.Client = 'AddressPingDemo,1,0';
