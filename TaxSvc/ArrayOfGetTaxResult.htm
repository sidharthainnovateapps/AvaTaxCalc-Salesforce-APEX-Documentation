TaxSvcSoap.ArrayOfGetTaxResult Class

The ArrayOfGetTaxResult class is a wrapper for an array of 0 or more GetTaxResult objects. Not meant to be instantiated directly, an ArrayOfGetTaxResult object is embedded in the ReconcileTaxHistoryResult object returned by calls to ReconcileTaxHistory.

global class ArrayOfGetTaxResult
{
     global TaxSvc.GetTaxResult[] GetTaxResult;
}
